Program VolumeTabung berfungsi untuk menghitung volume sebuah tabung berdasarkan jari-jari dan tinggi yang dimasukkan oleh pengguna. Cara kerjanya dimulai dari pemanggilan metode main(), di mana program menggunakan kelas Scanner untuk menerima input dari pengguna berupa jari-jari (r) dan tinggi (t). Nilai-nilai tersebut kemudian dikirim ke fungsi hitungVolume(), yang menghitung volume menggunakan rumus V=3.14×r×r×tV = 3.14 \times r \times r \times tV=3.14×r×r×t (πr²t). Fungsi ini mengembalikan hasil perhitungan ke metode utama, lalu program menampilkan hasil volume tabung ke layar. Dengan demikian, program ini memanfaatkan konsep fungsi dengan parameter untuk memisahkan logika perhitungan dari proses input-output, sehingga kode menjadi lebih rapi dan mudah dipahami.
